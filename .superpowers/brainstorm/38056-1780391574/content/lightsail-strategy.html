Lightsail — รู้ข้อจำกัดก่อน
หัวข้อ สถานะ รายละเอียด
รัน Docker Compose ได้ ✓ ติดตั้ง Docker บน Lightsail instance ได้ปกติ
Redis container ได้ ✓ เพิ่ม redis service ใน docker-compose.yml เลย
SSL / HTTPS ได้ ✓ Nginx + Let's Encrypt หรือ Lightsail cert ฟรี
Static IP ฟรี ✓ Attach static IP กับ instance ได้ฟรี 1 IP
Backup อัตโนมัติ จำกัด ⚠ Snapshot manual หรือ automatic $0.05/GB · ไม่ใช่ managed DB
Scale ขึ้นได้ไหม ต้อง stop ⚠ Upgrade plan ต้อง stop instance ~2-5 นาที downtime
Multi-instance SSE ต้องระวัง ⚠ Lightsail ไม่มี LB ดั้งเดิม · ถ้าขยายต้องใช้ Redis (มีอยู่แล้ว)
Connect กับ AWS services จำกัด ⚠ Lightsail อยู่ใน VPC แยก · ต้อง VPC peering ถ้าจะใช้ RDS/ElastiCache
S3 uploads ได้ ✓ Lightsail เรียก S3 API ได้ผ่าน IAM role · ไม่ต้อง VPC peering
แนะนำสำหรับตอนนี้: ใช้ Lightsail $10 ที่มีอยู่ได้เลย · เพิ่ม Redis service ใน docker-compose.yml · ใช้ local volume สำหรับ uploads ก่อน (S3 ทำทีหลังตอน Phase 2) · ค่าใช้จ่ายคงที่ $10/เดือน จนกว่าจะมีลูกค้าพอ · ข้อควรระวัง: ทำ snapshot backup ทุกอาทิตย์ เพราะ Lightsail ไม่มี managed DB · ถ้า disk เสีย ข้อมูลหาย · Phase 1 แค่ upgrade plan จาก $10 → $20 ใน Lightsail console ไม่ต้อง migrate เลย
